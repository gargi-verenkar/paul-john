# Paul John Caffeine — Shopify Theme

## Brand
- Colors: Cream (#FDF6EC), Brown (#4A2C0A), Orange (#C8651B)
- Tone: Premium, warm, South Indian coffee heritage

## Tech Stack
- Shopify Liquid (Dawn theme base)
- Vanilla JS, mobile-first CSS

## Sections to Build
- Homepage: hero slider, bestsellers, subscribe & save, testimonials, newsletter
- Product page with variant selectors (size + grind type)
- 5 collection pages, About, Wholesale, Contact

## Conventions
- Section files go in /sections/
- Snippet files go in /snippets/
- Always include {% schema %} blocks in sections
